2. 소놀봇-코덱스 맥버전

(맥버전은 테스트 횟수가 적습니다. 혹시 오류 있다면 알려주세요)



다음은 소놀봇-코덱스 윈도우버전 설치전에 갖추어야 할 사항입니다.
1) 윈도우 상에서 파이썬 설치  (https://www.python.org)
2) 윈도우 상에서 노드JS 설치 (https://nodejs.org)

3) WSL 설치

   1. PowerShell을 관리자 권한으로 실행
   2. 아래 명령 실행 → 재부팅
       wsl --install

4) 재부팅 후, WSL 접속하기
   1. PowerShell 실행
   2. 아래 명령 실행 
       wsl 


5) Node.js 설치 (4번 실행한 후 할것)
   (필수 도구 설치하기)
   sudo apt update
   sudo apt install -y curl

   (nvm 설치하기)
   curl o https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash
   source ~/.bashrc

6) Node 설치  (4번 실행하여, WSL 환경에서 할것)

   nvm install --lts
   nvm use --lts

   (잘되었는지 버전 확인)
   node -v
   npm -v


7) 파이썬 설치하기  (4번 실행하여, WSL 환경에서 할것)
   sudo apt update
   sudo apt install -y python3 python3-pip python3-venv

   (잘되었는지 버전 확인. 버전이 파이썬 3.12 이상이어야 합니다. 아니라면 AI 도움으로 3.12 이상 버전 갖추어 주세요) 
   python3 --version
   pip3 --version


8) Codex CLI 설치 (4번 실행하여, WSL 환경에서 할것)
   1. 설치
      npm i -g @openai/codex
   
   2. 이 명령 실행하여, 반드시 챗GPT 로그인까지 진행해두기 (그래야 소놀봇 코덱스 실행시 월 구독 플랜 내에서만 실행됨)
      codex

9) 영상에서 소개한 대로, 설치 폴더 내의 1.사전점검, 2.설치하기, 3.코덱스로그인, 4.컨트롤패널 파일을 순서대로 실행합니다.




** 소놀봇-코덱스 맥버전 설치하기 **

소스를 압축 풀고 적절한 위치로 이동한 상태에서, 해당 폴더 오른쪽 마우스 클릭하여, 터미널로 열기 실행합니다.

(터미널 진입후 다음 문장을 하나씩 실행합니다) 

# 1) Apple 개발 도구 설치
xcode-select --install

# 2) Homebrew 설치
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 3) nvm 설치
curl o https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash
source ~/.zshrc

# 4) Node LTS 설치
nvm install --lts
nvm use --lts
node -v
npm -v

# 5) pyenv 설치
brew update
brew install pyenv

# 6) pyenv zsh 설정
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc
echo '[[ -d $PYENV_ROOT/bin ]] && export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(pyenv init - zsh)"' >> ~/.zshrc
source ~/.zshrc

# 7) Python 3.12+ 설치
pyenv install 3.12.11
pyenv global 3.12.11
python --version
python3 --version
pip --version
pip3 --version

# 8) Codex CLI 설치 및 로그인
npm i -g @openai/codex
codex


(이제 제공해 드린 폴더에 마우스 오른쪽 클릭 후 터미널 열기로, 터미널 진입상태에서 아래 내용 순차적으로 진행합니다)
1. 실행 권한 부여 (한 문장씩 실행하세요)
chmod +x *.command
chmod +x scripts/*.sh
chmod +x mybot_autoexecutor.sh

2. 다운로드 격리(quarantine) 해제
xattr -dr com.apple.quarantine .

3. 설치파일들 실행 (한 문장씩 실행하세요)

./1.사전점검.command
./2.설치하기.command
./3.코덱스로그인.command
./4.컨트롤패널.command
